# Pinouts

#### Indicator LEDs
- Activity → A2
- Connected → A0
- Error → A1

#### HC-SR04 Ultrasonic Sensor
- VCC → 5V
- GND → GND
- Echo → D5
- Trig → D6

#### SD Card Module
- VCC → 5V
- GND → GND
- CS → D4
- SCK → D13
- MOSI → D11
- MISO → D12

#### DS3231 RTC Module
- VCC → 5V
- GND → GND
- SCL → AREF+2
- SDA → AREF+1

#### YF-S201 Flow Rate Sensor
- VCC → 5V
- GND → GND
- DATA → D2

#### SIM800L v2 GSM Module
- VCC → 5V
- GND → GND
- TXD → D7
- RXD → D8
